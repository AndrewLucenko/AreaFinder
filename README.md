# City Pulse Lab - NYC Event Locations Analysis

Пошук оптимальних локацій для проведення міських подій у Нью-Йорку.

## Структура проекту

```
AreaFinder/
├── data/
│   ├── raw/              # Сирі дані з API
│   └── processed/        # Оброблені дані
├── scripts/              # Python скрипти для обробки
├── output/               # Результати аналізу
├── visualization/        # Карти та графіки
└── README.md
```

## Датасети

- Parks: https://data.cityofnewyork.us/Recreation/Parks-Properties/enfh-gkve
- Restaurants: https://data.cityofnewyork.us/Health/DOHMH-New-York-City-Restaurant-Inspection-Results/43nn-pn8j
- Subway: https://data.ny.gov/Transportation/MTA-Subway-Stations/39hk-dx4f
